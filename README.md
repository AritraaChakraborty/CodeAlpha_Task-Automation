# File Organizer (Automation)

## Description :
A Python script designed to automate the organization of cluttered folders. It specifically scans a directory and moves all image files (specifically `.jpg`) into a dedicated folder.

## Features :
* Uses the `os` module to interact with the file system.
* Uses the `shutil` module for high-level file operations (moving files).
* Creates the destination folder automatically if it doesn't exist.

## Setup :
1. Create a folder named `my_files` in the same directory as the script.
2. Place some `.jpg` images inside `my_files`.
3. Run: `python automation.py`
